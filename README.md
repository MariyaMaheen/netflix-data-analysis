# 🎬 Netflix Data Analysis

A Python-based exploratory data analysis (EDA) project on the Netflix Movies & TV Shows dataset.

---

## 📊 What This Project Covers

| # | Task | Description |
|---|------|-------------|
| 1 | Descriptive Statistics | Mean, Median, Mode, Std Dev for key columns |
| 2 | Scatter Plot | Release Year vs Duration (Seaborn) |
| 3 | Box Plot | Movie duration — median, IQR, outliers |
| 4 | Histogram + KDE | Distribution of release years |
| 5 | Top 5 Countries | Countries with the most Netflix content |
| 6 | TV Shows Filter | Avg seasons of TV shows released after 2020 |
| 7 | Missing Value Imputation | Clean the entire dataset |

---

## 🚀 How to Run

**1. Clone the repo**
```bash
git clone https://github.com/your-username/netflix-analysis.git
cd netflix-analysis
```

**2. Install dependencies**
```bash
pip install -r requirements.txt
```

**3. Add the dataset**  
Place your `netflix_movies.csv` file in the project folder.

**4. Run the script**
```bash
python netflix_analysis.py
```

Charts will be saved automatically as `.png` files in the same folder.
